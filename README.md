# 📱 UFIXED - Aplikasi Pelaporan Kerusakan Barang

UFIXED adalah aplikasi berbasis React Native (Expo) yang digunakan untuk melaporkan kerusakan barang atau fasilitas secara digital dan terstruktur.

---

## 🚀 Teknologi yang Digunakan
- React Native
- Expo
- JavaScript
- Firebase (jika digunakan)

---

## 📌 Fitur Aplikasi
- Login pengguna
- Form pelaporan kerusakan
- Upload foto kerusakan
- Tracking status laporan
- Dashboard admin
